# NYS Lottery App

Electron desktop app for generating NYS lottery number picks.

## What Was Added

- Windows installer support using electron-builder (NSIS).
- GitHub Releases publishing flow for installer artifacts.
- In-app update checks using electron-updater from GitHub releases.

## One-Time Setup

1. Install Node.js LTS (if not already installed).
2. Install dependencies:

```powershell
npm install
```

1. Open package.json and set the build.publish values:

- owner: your GitHub username or org
- repo: your repository name

## Push Project To GitHub

```powershell
git init
git add .
git commit -m "Initial NYS Lottery app with installer and auto-update support"
git branch -M main
git remote add origin https://github.com/<owner>/<repo>.git
git push -u origin main
```

## Build Local Installer

```powershell
npm run dist
```

Installer output goes to release/.

## Publish A Release (and update feed)

1. Create a GitHub Personal Access Token with repo scope.
2. Set token in terminal session:

```powershell
$env:GH_TOKEN="your_token_here"
```

1. Bump version in package.json (for example 1.0.1).
2. Commit and push changes.
3. Build and publish release artifacts:

```powershell
npm run release
```

electron-builder uploads release assets to GitHub. Installed app instances use those releases for update checks.

## How Updates Work In The App

- Update checks run on app start (packaged builds only).
- If an update is available, user is prompted to download.
- When downloaded, user is prompted to restart and install.

## Notes

- Auto-updates do not run in development mode.
- The app must be installed from installer artifacts generated by electron-builder for updates to apply correctly.
